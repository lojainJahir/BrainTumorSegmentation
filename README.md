# **Brain Tumor Segmentation**

**This project uses deep learning to detect and segment brain tumors from MRI images.**

## **Tools**
- **Python**  
- **PyTorch**  
- **Google Colab**  

## **Project Overview**
**The model is trained on brain MRI images to identify tumor regions using image segmentation techniques.**

## **How to Run**
1. **Open the notebook in Google Colab.**
2. **Run the cells step by step.**
3. **Train the model and view the predictions.**
